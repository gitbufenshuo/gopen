# gopen
Golang writen 3D render library and some game-relative concepts.

# install

```
    # Find a directory which is not $GOPATH
    git clone https://github.com/gitbufenshuo/gopen
    cd gopen
    cd bins/gopeninit
    go run main.go
```

# try some examples

```
    cd gopen/example/basic
    go run main.go
```
